一、项目概述
  V21 是一个从零构建的白盒模块化中文认知引擎，用于错字纠正任务。
  所有模块从第一性原理设计，无黑盒依赖，全部可解释。
  总参数 ~40M，在 RTX 5070 12GB 上独立训练。

二、核心架构

  CharEmbed(3000字×256D) → P3属性栈(128D×126槽,14模块规则引擎)
  → P7交叉注意力(16头×16D, RMSNorm防爆)
  → P3-L属性联动(312头) → ABC三阶逻辑推理
  → 384D拼接Gate(base/abc/sent各128D独立通道)
  → Explore(384→1024→1024→1024) → Meta(1024→sigmoid)
  → P6解码(128头×1024→256, tied weights)
  → CE直出字

三、技术创新点

  1. 白盒模块化: 11个模块各有明确职责, 梯度流逐模块可追踪
  2. RMSNorm Q/K防爆: P7内部Q/K投影后归一化到单位球面,
     保方向信息, 彻底消灭Attention连乘爆炸
  3. 384D拼接Gate: 3个控制信号(base/abc/sent)各占128D独立通道,
     不竞争不截幅, 训练推理完全一致
  4. 128D P3属性体系: 14模块规则引擎, 126槽位3层分级,
     连接词+修饰等逻辑关系已激活
  5. 分层学习率: 基于梯度/参数比精确计算, P7×0.05, Gate×5.0
  6. abc_confidence: 12维ABC统计量替代prev_loss驱动Gate,
     消除训练推理gap
  7. NaN自动回滚: 梯度异常时从best checkpoint恢复, 不死训练

四、工程能力

  训练系统:
    - 7独立Adam优化器, 分层LR
    - 5层智能防护(ac检测/CE spike/渐进aux/NaN回滚/收敛停车)
    - P3属性GPU预缓存(0 CPU开销)
    - 梯度诊断: 逐模块grad/param ratio追踪

  数据处理:
    - 52K公开纠错数据字频/长度/覆盖度全分析
    - 贪心覆盖采样: 1500对覆盖5499字(89%) vs 随机20%
    - 脏数据过滤: 4360对(8.3%)长度不对齐/完全相同

  调试方法论:
    - gate消融实验 (有/无gate对比)
    - input scaling验证 (embed×3/5/10/20)
    - 双核辩论审计 (DeepSeek+千问, 9轮, 6个P0修复)
    - 逐模块梯度诊断 (定位sigmoid bias导致梯度死亡)

五、关键成果

  训练验证:
    - 5对: 完美背诵 ✅
    - 15对: E175 loss=0.001, 训练集78.6% ✅
    - 150对: CE正常下降, 无NaN ✅
    - 1500对: 训练中

  架构演进:
    V20(5M) → V21初期(10M) → V21完全体(40M)
    32头×4D → 16头×16D
    128D → 256D → 1024D
    P3 64D(18槽) → 128D(126槽)

  爆炸根因定位 (36小时内):
    sigmoid bias=2.0 → 梯度×0.1 → gate端5轮全死
    LayerNorm 128D同值→除零→NaN
    eval脚本维度不匹配→考试准确率0.3%是假象
    三个根因全部定位并修复
