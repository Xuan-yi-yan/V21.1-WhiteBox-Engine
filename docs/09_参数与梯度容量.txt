二、数据流维度 (30字句子为例)

  A句(30字) 输入
    │
    ▼ CharEmbed: 30×256 = 7,680
    │
    ├─→ P3属性: 30×128 = 3,840 (126槽, ~60槽已激活)
    │
    ▼ p7_input_proj: cat[30×256, 30×128] = 30×384 → 30×256
    │
    ▼ P7 CrossAttention:
    │   Q: 30×16头×16D = 7,680
    │   K/V: 全词表2900×16头×16D = 742,400
    │   注意力矩阵: 16头×30×2900 = 1,392,000
    │   word_out: 30×256 = 7,680
    │   sent_vec: 256
    │
    ├─→ word_out → ABC:
    │   A: 30×384→128 → 15类结构
    │   B: 128+128+128=384→128内容
    │   C: 128+5=133→48语气
    │
    ├─→ sent_vec → Gate:
    │   gate_input[12] = prev_loss[12] + abc[12] + sent[12]
    │   Explore: 12→1024→1024→1024 (~3.2M中间激活)
    │   Meta: 1024→sigmoid→gate[1024]
    │
    ▼ P6:
    │   sent_proj: 256→1024
    │   encoder: 1024→1024→1024 (×gate[1024])
    │   128头×Linear(1024→256): 128×256 = 32,768
    │   logits: 128×2900 = 371,200
    │
    ▼ B句(30字) 输出
